  • markdown2impress

markdown2impress

  • Perlで書かれた変換ツール
  • impress.js を使う
  • ページ区切りは<h1><h2>
  $ sudo cpan Data/Section/Simple.pm
  $ sudo cpan Text/Markdown.pm
  $ sudo cpan Text/Xslate.pm
  $ git clone git://github.com/yoshiki/markdown2impress.git
  $ cd markdown2impress
  $ bin/markdown2impress.pl README.md

mdslide

  • Rubyで書かれた変換ツール
  • サーバー機能付き
  • jQuery, slide.js を使う
  • ページ区切りが // 区切り
  $ gem install mdslide
  $ mdslide -i README.md -s
